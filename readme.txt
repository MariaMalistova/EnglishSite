В ходе работы над курсовой работой был создан сайт «Изучение иностранного языка». 
На данном сайте пользователь может изучить основные правила английского языка, найти полезные советы и ссылки, а так же пройти тест на знание языка. 
Сайт имеет адаптивную верстку и является удобным для использования на большинстве современных устройств.
